**Offline-first mobile app empowering 15+ field workers to monitor 1,200+ children**

An intelligent cross-platform survey collection platform built with Flutter and Hive DB, featuring:
- 📱 **Server-Driven UI** for flexible deployment without app updates
- 🔐 **Encrypted data sync** for secure offline operation
- 📊 **React.js admin dashboard** with dynamic survey builder & live impact analytics
- 👥 **Granular RBAC** for role-based access control
- ⚡ **40% reduction** in administrative validation overhead
